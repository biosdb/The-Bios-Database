This site only hosts **hashes**, not BIOS files themselves — use it to verify a file you already have, not to download one. Entries are cross-referenced against multiple independent sources (community wikis, [libretro-database](https://github.com/libretro/libretro-database), [Abdess/retrobios](https://github.com/Abdess/retrobios)) before being added, and unverified fields are left `null` rather than guessed.

Found a wrong hash, or have one to add? Contributions are welcome via pull request — see the [README](https://github.com/biosdb/The-Bios-Database#readme) for the data format.
