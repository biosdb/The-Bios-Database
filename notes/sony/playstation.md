The original PlayStation (PS1) BIOS is a 512 KiB boot ROM that differs by region and hardware revision. Every retail unit needs a matching BIOS dump to boot in an emulator.

**External resources:**

- [PlayStation (console) — Wikipedia](https://en.wikipedia.org/wiki/PlayStation_(console))
- [PSXDataCenter — BIOS listing](https://psxdatacenter.com/basics.html)
- [Emulation General Wiki — File Hashes](https://emulation.gametechwiki.com/index.php/File_hashes)

Most emulators expect the BIOS to be named `scph1001.bin` (NTSC-U) — see each emulator's own documentation for its exact expected filename.
